# ФОС дисциплины "Моделирование динамических систем"

Репозиторий содержит материалы для разработки рабочей программы дисциплины (РПД) и фонда оценочных средств (ФОС) по дисциплине "Моделирование динамических систем".

Дисциплина посвящена правилам и принципам применения дифференциальных уравнений для моделирования динамических процессов в разных предметных областях, включая задачи анализа данных, машинного обучения и проектирования интеллектуальных информационных систем.

## Назначение

Материалы репозитория используются для:

- проектирования структуры курса и результатов обучения;
- связывания содержания дисциплины с компетентностно-ролевой моделью КРМ 3.0;
- подготовки ФОС для оценки проектных и практических результатов студентов;
- обеспечения воспроизводимой методической базы для преподавателей и экспертов.

## Видение курса

Курс рассматривается как обзорно-практический: он должен дать студентам базовое понимание динамических моделей и показать, как эти модели применяются в прикладных кейсах с использованием современных вычислительных и ИИ-инструментов.

Ключевой фокус дисциплины:

- постановка задач моделирования динамических систем;
- дифференциальные уравнения как язык описания динамики;
- численное моделирование и анализ поведения систем;
- нелинейные динамические системы, устойчивость и хаос;
- связь моделей с данными, идентификация параметров и валидация;
- применение ИИ-инструментов для анализа, симуляции и интерпретации динамики.

## Целевые роли КРМ

Приоритетные роли, для которых дисциплина должна формировать полезные результаты обучения:

- ML Researcher (Исследователь МО) - продвинутый уровень;
- ML Engineer (Инженер МО) - базовый уровень;
- Data Analyst (Аналитик данных) - базовый уровень;
- AI CDO (Руководитель цифровой трансформации) - базовый уровень.

## Ожидаемые результаты этапов

Этап 0: подготовка инфраструктуры и базовых файлов репозитория.

Этап 1: разработка РПД: цель и задачи дисциплины, тематический план, пререквизиты, связь с КРМ 3.0, краткая система оценки результатов обучения.

Этап 2: разработка ФОС: паспорт, матрица компетенций, рубрики, шаблоны оценочных средств, правила peer-review и защиты.

Этап 3: подготовка материалов для публичной защиты результатов проектной работы.

## Источники

Основные исходные материалы расположены в папке `source`:

- техническое задание на разработку ФОС;
- постановка задания на проект РПД;
- КРМ 3.0;
- образцы и предметные материалы по дифференциальным уравнениям и нелинейным динамическим системам.

Файлы в `source` являются исходными материалами и не изменяются.
