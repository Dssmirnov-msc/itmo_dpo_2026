# 2. Постановка задачи лабораторной работы

## 2.1. Физическая задача

Рассматривается **одномерная задача теплопроводности** на отрезке $x \in [0, 1]$ с **неоднородным источником тепла**:

$$ \frac{\partial u}{\partial t} = \alpha \frac{\partial^2 u}{\partial x^2} + f(x,t), \quad (x,t) \in (0,1) \times (0,T]. $$

**Граничные условия** смешанные:

$$ u(0,t)=0, \quad \frac{\partial u}{\partial x}(1,t)=q(t). $$

**Начальное условие**:

$$ u(x,0)=u_0(x). $$

Где:

-   $\alpha = 0.01$ — коэффициент температуропроводности;
-   $f(x,t)$ — нестационарный источник тепла;
-   $q(t)$ — тепловой поток на правой границе;
-   $T = 1.0$ — конечное время.

## 2.2. Операторная постановка

Определим оператор $\mathcal{G}$, который ставит в соответствие **входным функциям** — источнику, граничным условиям и начальному условию — **решение** задачи:

$$ \mathcal{G}: (f(x,t), q(t), u_0(x)) \mapsto u(x,t). $$

**Цель работы** — обучить **нейронный оператор** DeepONet или FNO для аппроксимации оператора $\mathcal{G}$ и сравнить его с классическими численными методами.

# 3. Описание хода лабораторной работы

## 3.1. Генерация данных

Для генерации обучающей выборки необходимо задать случайные входные функции, решить задачу теплопроводности численным методом и сохранить пары «входные функции — поле решения».

## 3.2. Параметризация входных данных

**Источник тепла** $f(x,t)$ — сумма гауссиан:

$$ f(x,t) = \sum_{i=1}^{3} A_i \exp\left( -\frac{(x-\mu_i)^2}{2\sigma_i^2} -\frac{(t-\tau_i)^2}{2\gamma_i^2} \right). $$

Параметры:

-   $A_i \sim \mathcal{U}(-5, 5)$;
-   $\mu_i \sim \mathcal{U}(0.2, 0.8)$;
-   $\sigma_i \sim \mathcal{U}(0.05, 0.15)$;
-   $\tau_i \sim \mathcal{U}(0.2, 0.8)$;
-   $\gamma_i \sim \mathcal{U}(0.05, 0.15)$.

**Тепловой поток** $q(t)$ — сумма синусоид:

$$ q(t) = \sum_{k=1}^{2} B_k \sin(2\pi \omega_k t + \phi_k). $$

Параметры:

-   $B_k \sim \mathcal{U}(0.5, 2.0)$;
-   $\omega_k \sim \mathcal{U}(1, 5)$;
-   $\phi_k \sim \mathcal{U}(0, 2\pi)$.

**Начальное условие** $u_0(x)$ — сумма гауссиан:

$$ u_0(x) = \sum_{j=1}^{2} C_j \exp\left( -\frac{(x-\nu_j)^2}{2\delta_j^2} \right). $$

Параметры:

-   $C_j \sim \mathcal{U}(0.5, 2.0)$;
-   $\nu_j \sim \mathcal{U}(0.2, 0.8)$;
-   $\delta_j \sim \mathcal{U}(0.05, 0.15)$.

## 3.3. Численное решение

Для генерации эталонных решений используется **метод конечных разностей**.

**Сетка:**

-   пространство: $N_x = 100$ узлов, $h = 1/(N_x - 1)$;
-   время: $N_t = 100$ шагов, $\Delta t = T/N_t$;
-   шаг по времени должен удовлетворять условию устойчивости:

$$ \Delta t \leq \frac{h^2}{2\alpha}. $$

**Схема:**

-   использовать **неявную схему Кранка-Николсона** для повышения точности;
-   учитывать граничные условия на каждом временном шаге;
-   записывать решение во все узлы сетки.

**Объём выборки:**

-   обучающая выборка: 2000 примеров;
-   валидационная выборка: 200 примеров;
-   тестовая выборка: 100 примеров.

## 3.4. Базовый метод сравнения

Реализуйте **интерполяцию по ближайшему соседу** для сравнения:

$$ \widehat{u}(x,t) = u(x_i,t_j), \quad \text{где } (x_i,t_j) \text{ — ближайший узел сетки}. $$

Этот метод используется только как **ориентир** для оценки сложности задачи.

## 3.5. Нейронный оператор DeepONet

Реализуйте **DeepONet** (Deep Operator Network).

**Архитектура Branch Network:**

-   вход: дискретизация входных функций на сетке $N_{\text{input}} = 100$ точек;
-   выход: $p$-мерное скрытое представление;
-   архитектура: 3 полносвязных слоя с размерами $[512, 512, p]$.

**Архитектура Trunk Network:**

-   вход: координаты $(x,t)$;
-   выход: $p$-мерное скрытое представление;
-   архитектура: 3 полносвязных слоя с размерами $[512, 512, p]$.

**Выходной слой:**

$$ \widehat{u}(x,t) = \sum_{k=1}^{p} \text{branch}_k \cdot \text{trunk}_k. $$

**Входные данные для Branch:**

-   дискретизация $f(x,t)$ на фиксированной сетке по всем пространственно-временным узлам;
-   отдельные каналы для $q(t)$ и $u_0(x)$.

**Параметры:**

-   $p = 50$ — размер скрытого пространства;
-   активация: ReLU;
-   оптимизатор: Adam с `learning_rate = 10^{-4}`;
-   `batch size = 64`;
-   1000 эпох с ранней остановкой.

## 3.6. Нейронный оператор FNO

Реализуйте **Fourier Neural Operator**.

**Архитектура:**

1.  **Входной слой**: линейное отображение входных данных в скрытое пространство размерности $d_{\text{hidden}}$.
2.  **Серия Fourier-слоёв** ($L = 4$):
    -   преобразование Фурье $\mathcal{F}$;
    -   фильтрация в частотной области с сохранением $k_{\text{max}} = 12$ мод;
    -   обратное преобразование $\mathcal{F}^{-1}$;
    -   добавление линейного отображения исходных данных;
    -   применение активации GELU.
3.  **Выходной слой**: линейное отображение в скалярное поле.

**Параметры:**

-   $d_{\text{hidden}} = 32$;
-   $k_{\text{max}} = 12$;
-   оптимизатор: Adam с `learning_rate = 10^{-3}`;
-   scheduler: StepLR каждые 100 эпох.

## 3.7. Основные метрики качества

**Относительная** $L^2$-ошибка:

$$ \varepsilon_{L^2} = \frac{ \|u_{\text{pred}} - u_{\text{true}}\|_2 }{ \|u_{\text{true}}\|_2 }. $$

**Максимальная ошибка:**

$$ \varepsilon_{\max} = \max_{i,j} \left|u_{\text{pred}}(x_i,t_j)-u_{\text{true}}(x_i,t_j)\right|. $$

**Средняя абсолютная ошибка:**

$$ \varepsilon_{\text{MAE}} = \frac{1}{N_x N_t} \sum_{i,j} \left|u_{\text{pred}}(x_i,t_j)-u_{\text{true}}(x_i,t_j)\right|. $$

## 3.8. Дополнительные метрики

**Ошибка в частотной области:**

$$ \varepsilon_{\text{freq}} = \frac{ \|\mathcal{F}(u_{\text{pred}})-\mathcal{F}(u_{\text{true}})\|_2 }{ \|\mathcal{F}(u_{\text{true}})\|_2 }. $$

**Сохранение энергии:**

$$ E(t)=\int_0^1 u(x,t)\,dx, \quad \varepsilon_E = \max_t \frac{ \left|E_{\text{pred}}(t)-E_{\text{true}}(t)\right| }{ \max_t |E_{\text{true}}(t)| }. $$

# 5. Источники данных

Внешний набор данных не используется. Данные генерируются синтетически на основе одномерной задачи теплопроводности с неоднородным источником тепла, смешанными граничными условиями и случайным начальным условием.

## 5.1. Синтетические входные функции

Используются три типа входных функций:

-   источник тепла $f(x,t)$ как сумма гауссиан;
-   тепловой поток $q(t)$ как сумма синусоид;
-   начальное условие $u_0(x)$ как сумма гауссиан.

## 5.2. Эталонное численное решение

Эталонные решения генерируются методом конечных разностей с использованием неявной схемы Кранка-Николсона. Эталон используется для обучения, валидации, тестирования и сравнения нейронных операторов с классическим численным подходом.

# 7. Методические рекомендации для преподавателя

## 7.1. Подготовка задания

-   Можно использовать упрощённую версию задачи: $f(x,t)=0$ или $q(t)=0$;
-   для ускорения обучения можно уменьшить разрешение сетки до $N_x = 50$, $N_t = 50$;
-   можно предложить студентам сравнить результат с PINNs;
-   для бонусных баллов можно попросить реализовать Operator Network with Attention или Transformer-based Operator.

## 7.2. Методические рекомендации для студентов

-   Начните с реализации численного метода: это важно для генерации данных и проверки моделей;
-   сначала реализуйте простую версию модели, например DeepONet с $p = 20$, затем масштабируйте;
-   для FNO можно использовать готовую библиотеку `neuraloperator`, а затем реализовать собственную версию;
-   обратите внимание на нормализацию входных данных: это критически важно для сходимости;
-   для оценки неопределённости используйте FNO, так как DeepONet не поддерживает дропаут.
