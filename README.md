## 1. О дисциплине

**Цель дисциплины.** Сформировать у обучающихся способность применять современные ИИ-решения при создании динамических моделей сложных систем на основе дифференциальных уравнений, данных и вычислительных методов.

**Структура дисциплины.** Текущая версия РПД включает пять модулей:

1.  Моделирование сложных систем.
2.  Методология трансляции (проектирование модели).
3.  Физико-информированные нейронные сети (PINN).
4.  Нейронные операторы.
5.  Физико-обоснованное суррогатное моделирование сложных динамических систем.

**Целевые роли КРМ:** ML Researcher, ML Engineer, Data Analyst.

**Итоговая форма контроля:** проект.

Полное описание дисциплины, тематический план, компетенции и планируемые результаты обучения приведены в [РПД][rpd].

## 6. Итоговая оценка

Текущая рабочая логика ФОС использует 10-балльную шкалу в отдельных лабораторных работах. Общая формула итоговой оценки пока не утверждена.

Предварительная рамка:

-   текущий и рубежный контроль должен складываться из лабораторных работ и контрольной работы по формализации;
-   промежуточная аттестация должна проходить в форме проекта;
-   проект должен интегрально проверять модель, код, вычислительный эксперимент, интерпретацию результатов, ограничения и обоснование выбранных ИИ-решений.

**Что ещё нужно определить команде:**

-   веса модулей и лабораторных работ;
-   минимальные пороги допуска к проекту;
-   правила пересдачи и доработки КИМ;
-   рубрику итогового проекта;
-   правила использования генеративного ИИ при выполнении работ.
