# Примеры решений контрольной работы Методология трансляции (проектирование модели)

## Решение варианта 1

### Задача 1
1. **Алгебраический баланс объемов**: Для сохранения постоянства объемов воды в резервуарах суммарный приток должен быть строго равен оттоку.  
   Рассмотрим резервуар 1: в него втекает подпиточная вода $Q_0$ и циркуляционный поток из второго резервуара $Q$. Вытекает дренажный сброс $Q_d$, испарение $E$ и поток во второй резервуар $Q$.  
   Поскольку внутренние потоки обмена сбалансированы ($Q$), баланс внешних потоков дает искомое условие:  
   **$$Q_0 = Q_d + E$$**

2. **Дифференциальный баланс массы солей**: Учитываем, что испаряется только чистая вода (поток солей с паром равен нулю), а перемешивание мгновенное. Изменение массы соли $M_i = V_i C_i$ в каждой камере равно разности притока и оттока солей:
   * Резервуар 1: $\frac{d(V_1 C_1)}{dt} = Q_0 C_0 + Q C_2 - Q C_1 - Q_d C_1$
   * Резервуар 2: $\frac{d(V_2 C_2)}{dt} = Q C_1 - Q C_2$

   Учитывая константность объемов ($V_1, V_2 = \text{const}$), делим на них левые части уравнений:  
   **$$\frac{dC_1}{dt} = \frac{Q_0 C_0 + Q C_2 - (Q + Q_d)C_1}{V_1}$$**  
   **$$\frac{dC_2}{dt} = \frac{Q}{V_2}(C_1 - C_2)$$**

### Задача 2
1. Подставим замену переменных $K = K_c k$ и $t = t_c \tau$ в исходное дифференциальное уравнение:  
   $$\frac{K_c}{t_c} \frac{dk}{d\tau} = A K_c^\alpha k^\alpha - \delta K_c k$$
2. Умножим обе части на комплекс $\frac{t_c}{K_c}$ для нормализации производной:  
   $$\frac{dk}{d\tau} = (A t_c K_c^{\alpha-1}) k^\alpha - (\delta t_c) k$$
3. Избавимся от параметров, потребовав, чтобы коэффициенты при членах в правой части обратились в единицу:
   * $\delta t_c = 1 \implies \mathbf{t_c = \frac{1}{\delta}}$
   * $A t_c K_c^{\alpha-1} = 1 \implies \frac{A}{\delta} K_c^{\alpha-1} = 1 \implies K_c^{1-\alpha} = \frac{A}{\delta} \implies \mathbf{K_c = \left(\frac{A}{\delta}\right)^{\frac{1}{1-\alpha}}}$
4. Запишем финальный безразмерный вид системы и начальных условий:  
   **$$\frac{dk}{d\tau} = k^\alpha - k, \quad k(0) = \frac{K_0}{K_c} = K_0 \left(\frac{\delta}{A}\right)^{\frac{1}{1-\alpha}}$$**

## Решение варианта 2

### Задача 1
1. **Баланс субстрата $S$**: Концентрация органики внутри проточного аэротенка изменяется за счет входящего внешнего потока, вымывания несвязанного субстрата и его биопотребления бактериями по закону Моно:  
   $$V \frac{dS}{dt} = Q S_0 - Q S - V \frac{\mu_{max} S}{K_s + S} X$$  
   Разделив на постоянный объем реактора $V$:  
   **$$\frac{dS}{dt} = \frac{Q}{V}(S_0 - S) - \frac{\mu_{max} S X}{K_s + S}$$**

2. **Баланс биомассы бактерий $X$**: Внешний приток активного ила отсутствует. Популяция размножается с эффективностью переработки $Y$ и вымывается из системы:  
   $$V \frac{dX}{dt} = V Y \frac{\mu_{max} S}{K_s + S} X - Q X$$  
   После деления на объем $V$:  
   **$$\frac{dX}{dt} = \frac{Y \mu_{max} S X}{K_s + S} - \frac{Q}{V} X$$**

### Задача 2
1. Подставим безразмерные подстановки $x = A u$ и $t = \frac{\tau}{r}$ в исходную динамическую модель:  
   $$A r \frac{du}{d\tau} = r A u \left(1 - \frac{A u}{K}\right) - \frac{B A^2 u^2}{A^2 + A^2 u^2}$$
2. Разделим обе части на масштабирующий множитель производной $A r$:  
   $$\frac{du}{d\tau} = u \left(1 - \frac{A}{K} u\right) - \frac{B}{A r} \frac{u^2}{1 + u^2}$$
3. Сгруппируем оставшиеся размерные константы в минимальный набор безразмерных параметров:
   * **$$q = \frac{A}{K}$$** — отношение полунасыщения хищника к емкости среды;
   * **$$p = \frac{B}{A r}$$** — безразмерный потенциал трофического давления хищника.
4. Запишем каноническую безразмерную форму уравнения:  
   **$$\frac{du}{d\tau} = u (1 - q u) - \frac{p u^2}{1 + u^2}$$**

## Решение варианта 3

### Задача 1
1. Составим базовое балансовое уравнение для численности инфицированных слухом агентов. Приток формируется за счет парных контактов восприимчивых с носителями (закон действующих масс), отток — за счет забывания информации с эффектом насыщения:  
   $$\frac{dI}{dt} = \beta S I - \frac{\gamma I}{1 + a I}$$
2. Так как популяция изолирована, выполняется закон сохранения $S(t) + I(t) = N \implies S = N - I$.
3. Подставим выражение для $S$ в исходный баланс, исключая лишнюю переменную состояния:  
   **$$\frac{dI}{dt} = \beta (N - I) I - \frac{\gamma I}{1 + a I}$$**

### Задача 2
1. Введем безразмерные масштабы через подстановку $I = I_c y$ и $t = t_c \tau$:  
   $$\frac{I_c}{t_c} \frac{dy}{d\tau} = \frac{\beta S_0 I_c y}{1 + a I_c y} - \gamma I_c y$$
2. Разделим обе части на $I_c$ и умножим на временной масштаб $t_c$:  
   $$\frac{dy}{d\tau} = \frac{\beta S_0 t_c y}{1 + (a I_c) y} - (\gamma t_c) y$$
3. Следуя указаниям задания, выберем масштабы из соображений упрощения структуры уравнения:
   * Насыщение знаменателя: $a I_c = 1 \implies \mathbf{I_c = \frac{1}{a}}$
   * Темп элиминации: $\gamma t_c = 1 \implies \mathbf{t_c = \frac{1}{\gamma}}$
4. Подставим выбранные масштабы в коэффициент числителя:  
   $$\beta S_0 t_c = \frac{\beta S_0}{\gamma} = R_0$$
5. Итоговая безразмерная форма имеет вид:  
   **$$\frac{dy}{d\tau} = \frac{R_0 y}{1 + y} - y$$**  
   *Физический смысл параметра $R_0$*: Базовое репродуктивное число эпидемии, показывающее среднее число заражений, производимых одним инфицированным в полностью восприимчивой среде за время болезни.

## Решение варианта 4

### Задача 1
1. Перейдем от масс веществ к концентрациям с помощью геометрических соотношений объемов камер: $C_b = M_b / V_b$ и $C_t = M_t / V_t$.
2. **Баланс в крови (Камера 1)**: Масса препарата увеличивается за счет внешней капельницы с интенсивностью $R$, уменьшается из-за пассивной мембранной диффузии в ткани (пропорциональна разности концентраций) и почечного клиренса:  
   $$V_b \frac{dC_b}{dt} = R - k(C_b - C_t) - Cl \cdot C_b$$  
   Перенося объем в правую часть:  
   **$$\frac{dC_b}{dt} = \frac{R}{V_b} - \frac{k}{V_b}(C_b - C_t) - \frac{Cl}{V_b}C_b$$**

3. **Баланс в тканях (Камера 2)**: Изменение массы лекарства происходит исключительно за счет диффузионного притока-оттока из кровеносного русла:  
   $$V_t \frac{dC_t}{dt} = k(C_b - C_t)$$  
   После деления на объем $V_t$:  
   **$$\frac{dC_t}{dt} = \frac{k}{V_t}(C_b - C_t)$$**

### Задача 2
1. Подставим безразмерные переменные $X = X_c x$ и $t = t_c \tau$ в балансовое уравнение автокатализа:  
   $$\frac{X_c}{t_c} \frac{dx}{d\tau} = A - (B + 1)X_c x + C Y_0 X_c^2 x^2$$
2. Умножим уравнение на комплекс $\frac{t_c}{X_c}$:  
   $$\frac{dx}{d\tau} = \frac{A t_c}{X_c} - (B + 1)t_c x + (C Y_0 X_c t_c) x^2$$
3. Зададим условия нормировки параметров в единицу согласно техническому заданию:
   * Свободный член: $\frac{A t_c}{X_c} = 1 \implies t_c = \frac{X_c}{A}$
   * Нелинейный член: $C Y_0 X_c t_c = 1 \implies C Y_0 X_c \left(\frac{X_c}{A}\right) = 1 \implies X_c^2 = \frac{A}{C Y_0}$
4. Находим аналитические выражения для масштабов системы:  
   **$$X_c = \sqrt{\frac{A}{C Y_0}}$$**  
   **$$t_c = \frac{1}{\sqrt{A C Y_0}}$$**
5. Подставим найденные константы в оставшийся линейный коэффициент при переменной $x$:  
   $$\gamma = (B + 1)t_c = \frac{B + 1}{\sqrt{A C Y_0}}$$
6. Запишем финальное безразмерное обыкновенное дифференциальное уравнение:  
   **$$\frac{dx}{d\tau} = 1 - \gamma x + x^2$$**
